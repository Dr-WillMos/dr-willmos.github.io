关于我

Q：博主是？
A：一个大学生，就读于杭州电子科技大学，2024级网络工程。

Q：为什么会搭建这个博客？
A：我正在向无纸化学习方向转变，考虑到我需要在不同设备中访问笔记和互联网开放、共享、去中心化的精神，我购买了域名并使用HEXO搭建了此博客，如果能帮到某个读者，倍感荣幸。

Q：你主要写些什么内容？
A：主要分享：

  • Java Web 学习笔记（Servlet、Tomcat、Spring Boot 等）
  • 前端基础知识（HTML、CSS、JavaScript）
  • 项目实战与开发经验
  • 一些对学习、生活的思考

Q：目前在学什么？
A：正在深入学习：

  • Maven 项目管理
  • Servlet 与 JSP 技术
  • Web 应用的整体开发流程
  • 以及一些算法与数据结构的基础

Q：博客更新频率？评论生态管理？
A：不定期更新，毕竟质量大于数量。本博客评论系统使用了Waline开源项目，已接入邮箱通知，如有违规评论我会在第一时间清除，本博客对有关意识形态，无理由谩骂的评论采取零容忍态度。

Q：未来的目标是什么？
A：保持学习，保持进步，保持思考。

Q：这个博客的意义是什么？
A：对我来说，博客不仅是知识的记录本，也是与过去的自己对话的地方。
写作让我更清楚自己到底懂了什么、还缺什么。

意义是没有意义的，没人能用它来绑架我。

A：不，这个博客具有很强的私人性质，你可以理解为一个可以在线访问的私人笔记，我有时候为了易读会用AI美化我碎片化的笔记。

自我和解

Q： 你怎么处理看到别人进步时，心里滋生的嫉妒？
A：首先，这其实是一个关于选择的问题——你如何分配自己的时间与精力，最终决定了你能成为什么样的人。举个例子，健身看似只是肌肉在运动刺激下吸收蛋白质并增长，这是投入与回报相对明确的事；但人生中大多数事情并非如此简单，付出没有回报的事情多了去了。其次，因选择不同，我们走在各自的路上，用自己的尺度去衡量别人的脚步，往往只会带来自卑或自负。最后，生活是你自己的，每一次嫉妒其实都是内耗，会消耗你宝贵的精力。把精力用在如何过好自己的生活上，才是更实在的选择——放过自己，不要为不必要的不愉快浪费时光。

Q：你怎么看待你们这一代的迷茫？
A：身处迷雾的森林之中，隐隐约约看见几条路，你不知道该选哪条，没有地图，没有指南针，甚至没有前人走过的脚印，只有嘶吼的风和盘旋的落叶，你只能凭着有限且真假参半的信息做出选择，并独自承这不得不做出的选择带来的后果。最后你和那些别的路上的人交流，你会也许会发现你选对了，或者你选错了，又或者你发现哪一条路都是不对的。院子里的狗羡慕流浪猫的自由，流浪猫羡慕院子里的狗的温饱，我们总是喜欢去美化那一条自己走过的路，但实际上没有万全之策，走好自己已经选择的路，才是最有用的，毕竟决策也是努力的一部分。

Q：你打算谈恋爱吗?
A：我不知道，我不确定有没有爱人的能力，毕竟我自爱的能力本身就值得怀疑。我看过的谎言和怨气太多，我没有理由相信情比金坚，以致于我自己都不敢迈出那一步。

Q： 你自己有心理问题吗？
A：是的，我正在等待治愈，鉴于大脑的保护机制，我会遗忘那些让我感到痛苦的经历，但也许会在某个午后想起来，也许不会。我很多时候用学习来填充我的生活，将心理问题压制在心底，这也是我在表面上看是一个正常人的原因。
